# EOE 看图猜词

AI先帮我写一版，后面我再自己改
一个基于 Web 技术开发的看图猜词游戏，支持自定义题库管理。

## 功能特点

- 🎮 **游戏玩法**：通过图片和问题猜测答案，支持提示和放弃功能
- 📚 **题库管理**：支持创建、编辑、导入、导出题库
- 🎵 **音效系统**：内置背景音乐和音效，可调节音量
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🚀 **自动加载**：自动加载初始题库，无需手动导入

## 运行说明

### 方法一：直接运行可执行文件

1. 下载 `eoeguass.exe` 文件
2. 双击运行即可

### 方法二：本地 HTTP 服务器

1. 克隆或下载项目到本地
2. 打开终端，进入项目目录
3. 运行以下命令启动本地服务器：
   
   ```bash
   python -m http.server 8000
   ```
   
4. 在浏览器中访问 `http://localhost:8000`

## 使用指南

### 开始游戏

1. 点击「开始游戏」按钮
2. 选择一个题库
3. 查看图片和问题，输入答案
4. 点击「回答」按钮检查答案
5. 可使用「提示」或「放弃」功能
6. 完成所有题目后查看统计结果

### 管理题库

1. 点击「管理题库」按钮
2. 可选择「新增题库」或「导入题库」
3. 编辑题库时，可添加、修改、删除题目
4. 支持导出题库为 JSON 文件

### 设置

1. 点击「设置」按钮
2. 可调节背景音乐和音效音量

## 开发说明

### 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **存储**：LocalStorage
- **音效**：Web Audio API

### 核心功能实现

- **题库管理**：使用 `QuestionBank` 类处理题库的增删改查
- **游戏逻辑**：实现了答题、计分、提示、放弃等功能
- **页面切换**：通过 CSS 类控制页面显示隐藏
- **自动加载**：在 `DOMContentLoaded` 事件中自动加载初始题库
