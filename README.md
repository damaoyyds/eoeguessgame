## 🎯 游戏简介

EOE 谐音梗挑战是一款有趣的益智游戏，玩家需要根据图片和提示猜测对应的谐音梗答案。游戏包含多期题库，每期都有精心设计的题目，考验玩家的联想能力和反应速度。

## ✨ 功能特点

- 🎮 **丰富的游戏内容**：包含多期题库，每期15道题目
- 🖼️ **精美图片**：每道题目都有对应的图片提示
- 🎵 **沉浸式音效**：内置背景音乐和交互音效
- 💡 **提示系统**：遇到困难时可使用提示功能
- 📱 **响应式设计**：适配不同屏幕尺寸，支持移动端
- 💾 **本地存储**：自动保存游戏进度和设置

## 🚀 快速开始

### 方法一：本地 HTTP 服务器（推荐）

1. 克隆或下载项目到本地
2. 打开终端，进入项目目录
3. 运行以下命令启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问 `http://localhost:8000`

### 方法二：直接打开

直接双击 `index.html` 文件在浏览器中打开（部分浏览器可能会限制本地文件的功能）

## 📁 项目结构

```
├── banks/          # 题库文件
│   ├── 第一期_updated.json
│   └── 第二期_updated.json
├── images/         # 图片资源
│   ├── s1/         # 第一期题目图片
│   ├── s2/         # 第二期题目图片
│   ├── icon.png    # 图标
│   └── wallpaper.webp  # 背景图片
├── sounds/         # 音效文件
│   ├── bgm.mp3     # 背景音乐
│   ├── clear.wav   # 清除音效
│   ├── click.wav   # 点击音效
│   ├── lose.wav    # 失败音效
│   └── win.wav     # 胜利音效
├── index.html      # 主页面
├── script.js       # 游戏逻辑
├── styles.css      # 样式文件
└── front.ttf       # 字体文件
```

## 🎮 游戏玩法

1. **选择题库**：在首页选择要挑战的题库
2. **查看题目**：每道题目会显示一张图片和对应的问题
3. **输入答案**：在输入框中输入你猜测的答案
4. **提交答案**：点击「回答」按钮检查答案是否正确
5. **使用提示**：遇到困难时可点击「提示」按钮获取帮助
6. **放弃题目**：实在想不出来可以点击「放弃」跳过此题
7. **查看结果**：完成所有题目后，会显示你的得分和用时

## 🔧 技术实现

### 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **存储**：LocalStorage
- **音效**：Web Audio API
- **图片**：WebP 格式（优化加载速度）

### 核心功能

- **题库管理**：支持多期题库的加载和切换
- **游戏逻辑**：实现了答题、计分、提示、放弃等核心功能
- **音效系统**：背景音和交互音效的播放控制
- **动画效果**：答题正确/错误时的反馈动画
- **本地存储**：保存游戏进度和用户设置

## 📝 自定义题库

### 题库格式

题库文件为 JSON 格式，位于 `banks/` 目录下。每个题库文件包含以下结构：

```json
{
  "id": "bank1",
  "name": "第一期",
  "questions": [
    {
      "id": 1,
      "question": "问题描述",
      "answer": "正确答案",
      "hint": "提示信息",
      "image": "s1/question_1.webp"
    },
    // 更多题目...
  ]
}
```

### 添加新题库

1. 在 `banks/` 目录下创建新的 JSON 文件
2. 按照上述格式添加题目
3. 在 `images/` 目录下创建对应期数的文件夹（如 `s3/`）
4. 将题目对应的图片放入该文件夹
5. 刷新游戏页面，新题库会自动显示

## 🎨 界面说明

- **首页**：显示游戏标题、开始按钮和设置选项
- **题库选择**：展示可用的题库列表
- **游戏界面**：包含图片、问题、输入框和操作按钮
- **结果页面**：显示游戏得分、用时和重新开始选项
- **设置面板**：可调节音量和其他游戏设置

## 🔊 音效说明

- **背景音乐**：游戏过程中循环播放
- **点击音效**：点击按钮时播放
- **胜利音效**：答题正确时播放
- **失败音效**：答题错误时播放
- **清除音效**：清除输入框时播放
